The System Journey: Program Workflow

1. User Access & Onboarding

Access system via web/app; Account creation or anonymous login; Data privacy consent.

2. Main Menu & Navigation

Choose: File New Report, Check Status, Access Resources/FAQ, or Chatbot help.

3. Incident Submission & Tracking

Resident inputs details (guided by chatbot); System auto-assigns Case ID and sends notification.

4. Staff Processing & Action

Barangay staff review dashboard; Classify report; Forward to Lupong Tagapamayapa or PNP.

5. Real-Time Updates & Resolution

System sends status changes (e.g., "Pending Mediation," "Resolved"); Final report archived.

6. Data Collection & Analytics

Secure storage of blotter data; System runs AI to identify crime trends and high-risk zones.

7. Outcome Monitoring & Governance

Officials view Admin dashboard analytics; Generate automated compliance reports for LGUs.
